版本说明 ​

版本格式 ​

MAJOR.MINOR.PATCH（主版本.次版本.修订版本）

  • MAJOR：不兼容的 API 变更
  • MINOR：向后兼容的功能新增
  • PATCH：向后兼容的问题修复

变更类型 ​

  • 新增：新功能
  • 修复：Bug 修复
  • 改进：性能优化、代码重构等
  • 废弃：即将移除的功能
  • 删除：已移除的功能
  • 安全：安全修复
