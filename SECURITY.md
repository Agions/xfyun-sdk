# Security Policy

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.5.x | ✅ 当前支持 |
| 1.4.x | ✅ 安全修复 |
| < 1.4 | ❌ 不再支持 |

## 报告安全漏洞

我们非常重视安全问题。如果你发现安全漏洞，请：

### 首选：私钥报告

发送电子邮件至 `1051736049@qq.com`，包含：
- 漏洞描述
- 复现步骤
- 影响范围
- 建议的修复方案（可选）

### 备选：GitHub Security Advisory

1. 访问 https://github.com/Agions/xfyun-sdk/security/advisories/new
2. 选择 "Report a vulnerability"
3. 填写详细信息

## 响应时间

- **严重漏洞**：24 小时内响应
- **高严重性**：72 小时内响应
- **中等严重性**：1 周内响应
- **低严重性**：2 周内响应

## 修复流程

1. 确认漏洞
2. 评估影响范围
3. 开发修复
4. 测试验证
5. 发布安全补丁
6. 更新此文档

## 安全最佳实践

### 对于使用者

- 始终使用最新版本
- 不要在前端代码中暴露 API Secret
- 使用 HTTPS 连接
- 定期更新依赖

### 对于贡献者

- 不要提交包含敏感信息的代码
- 使用 `npm audit` 检查依赖漏洞
- 遵循安全编码规范
